Bagel.ai is an AI content generator that helps you write original, quality, content faster. It is powered by cutting-edge technologies like ChatGPT and OpenAI.

== Description ==

Bagel.ai helps you write better content faster. You can automatically generate full blog posts, outlines, summarize text, expand on sentences, and even generate ideas on what to write.

== Features ==

* Ask Bagel.ai to write a full blog post for you
* Start a sentence and ask Bagel.ai to expand on it
* Ask Bagel.ai to summarize anything, such as text, articles, or documents
* Ask Bagel.ai to generate a blog post outline for you
* Ask Bagel.ai to generate ideas for you, on what to write and even how to write it
* Ask Bagel.ai to optimize your content for SEO

== Installation ==

1. Upload `bagel-ai.zip` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Create a free account on [bagel.ai](https://bagel.ai "bagel.ai") to get an API key.
4. In your WordPress Admin, go to the Bagel.ai plugin settings page.
5. Click on 'Manually enter a new API Key'.
6. Click "Connect API Key".
7. Your plugin is now ready to use!

This WordPress plugin integrates with bagel.ai and OpenAI in order to generate content, every time you create or edit content.
